EggRoll 개인정보 처리방침

최종 업데이트: 2025-10-29

1. 수집하는 정보

1.1 라이선스 관리를 위한 정보

  • 이메일 주소: Google OAuth를 통해 수집
  • 라이선스 키: 사용자가 입력
  • 수집 목적: Premium 기능 활성화 및 라이선스 관리

1.2 디바이스 식별 정보 (무제한 복제 방지)

  • CPU 코어 수: navigator.hardwareConcurrency
  • 화면 해상도: screen.width × screen.height × colorDepth
  • GPU 정보: WebGL 렌더러 정보
  • 렌더링 특성: Canvas fingerprint
위 하드웨어 정보는 해시 처리되어 디바이스 ID로 변환됩니다.
원본 하드웨어 정보는 서버에 저장되지 않으며, 해시된 디바이스 ID만 저장됩니다.

1.3 사용 로그

  • 라이선스 검증 시간
  • 활성화/비활성화 기록
  • 수집 목적: 부정 사용 방지 및 문제 해결

2. 데이터 저장 위치

  • Firebase Firestore: 라이선스 데이터 (HTTPS 암호화 전송)
  • Google Drive AppDataFolder: 라이선스 동기화 (사용자 본인의 Drive 공간)
  • Chrome Storage Local: 캐시 데이터 (로컬에만 저장)

3. 데이터 사용 목적

  1. 라이선스 활성화 및 검증
  2. 기기 간 라이선스 동기화
  3. 디바이스별 활성화 제한 (최대 2대)
  4. 부정 사용 및 무단 복제 방지

4. 데이터 공유

  • 제3자에게 판매하거나 공유하지 않습니다
  • 광고 목적으로 사용하지 않습니다
  • Google Drive 및 Firebase는 데이터 저장 목적으로만 사용됩니다

5. 데이터 보안

  • 모든 데이터는 HTTPS를 통해 암호화 전송됩니다
  • Firebase Firestore 보안 규칙이 적용되어 있습니다
  • Google Drive AppDataFolder는 확장 프로그램만 접근 가능합니다

6. 사용자 권리

  • 라이선스 관리 페이지에서 등록된 기기를 확인하고 해제할 수 있습니다
  • Google 계정 연결을 해제하면 Drive 데이터가 자동으로 삭제됩니다
  • 데이터 삭제를 요청하실 수 있습니다

7. 정책 변경

본 개인정보 처리방침이 변경될 경우, 확장 프로그램 업데이트를 통해 공지하며, 변경 사항은 이 페이지에 게시됩니다.
