개인정보 처리방침

최종 업데이트: 2025년 10월 29일

EggRoll은 코코포리아(CCFolia, ccfolia.com) 플랫폼에 향상된 기능을 제공하는 크롬 확장 프로그램입니다. 저희는 귀하의 개인정보를 보호하기 위해 최선을 다하고 있습니다.

1. 수집하는 정보

1.1 라이선스 정보

프리미엄 라이선스를 활성화할 때 다음 정보를 수집합니다:

  • 라이선스 키: 라이선스 검증을 위해 사용
  • 이메일 주소: 라이선스 소유자 확인을 위해 사용
  • 기기 ID: 기기를 식별하기 위해 무작위로 생성된 UUID

기기 ID는 간단한 무작위 식별자로, 로컬에 저장됩니다. 개인정보나 하드웨어 사양은 전혀 포함하지 않습니다.

1.2 로컬 설정

확장 프로그램의 환경설정은 사용자 기기에 로컬로 저장됩니다:

  • 테마 설정 (색상, 글꼴)
  • 커스텀 매크로

이 데이터는 기기에만 저장되며, Chrome 동기화를 활성화한 경우에만 동기화됩니다. 저희는 이 데이터에 접근할 수 없습니다.

2. 정보 사용 목적

라이선스 정보는 다음 목적으로만 사용됩니다:

  • 유효한 라이선스 보유 여부 확인
  • 2대 기기 활성화 제한 적용
  • 기기 활성화 관리 기능 제공

3. 데이터 저장

라이선스 데이터는 Firebase Cloud Firestore에 다음 보안 조치와 함께 저장됩니다:

  • HTTPS 암호화
  • 인증된 요청만 허용
  • 요청 빈도 제한(Rate limiting)

로컬 설정은 Chrome의 storage API를 사용하여 저장되며, Chrome 동기화가 활성화되지 않은 경우 기기 밖으로 전송되지 않습니다.

4. 데이터 공유

저희는 귀하의 데이터를 제3자에게 판매, 거래, 공유하지 않습니다.

귀하의 데이터는 라이선스 검증 및 확장 프로그램 기능 제공 목적으로만 사용됩니다.

5. 귀하의 권리

다음과 같은 권리를 행사하실 수 있습니다:

  • 라이선스를 사용 중인 기기 목록 확인
  • 기기 활성화를 해제하여 활성화 슬롯 확보
  • 확장 프로그램을 제거하여 모든 로컬 데이터 삭제

6. 제3자 서비스

이 확장 프로그램은 라이선스 저장을 위해 Firebase(Google)를 사용합니다. Firebase의 개인정보 처리방침은 firebase.google.com/support/privacy에서 확인하실 수 있습니다.
